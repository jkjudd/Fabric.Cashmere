Module Import

import {InputModule} from '@healthcatalyst/cashmere';

Directives

HcFormFieldComponent

Container for form fields that applies Cashmere styling and behavior

Selector: hc-form-field

InputDirective

Directive that allows a native input to work inside a HcFormFieldComponent

Selector: [hcInput]

Properties
Name Description
@Input()

disabled: boolean

Disables the input element.
@Input()

id: string

Element id.
@Input()

placeholder: string

Hint displayed within the input and disappears on input.
@Input()

readonly: boolean

Sets input element as readonly.
@Input()

required: boolean

Sets required attribute.
@Input()

type: string

Input type of the element.
@Input()

value: string

Sets value of the input element
Methods
focus()

Sets the focus on the input element

HcErrorComponent

Single error to be shown under HcFormFieldComponent

Selector: hc-error

HcPrefixDirective

Position an element at the front of HcFormFieldComponent

Selector: [hcPrefix]

HcSuffixDirective

Position an element at the end of HcFormFieldComponent

Selector: [hcSuffix]

HcLabelComponent

Label for HcFormFieldComponent

Selector: hc-label
