Directives

PaginationComponent

The pagination control enables the user to navigate across paged content. Although commonly used with tables and data grids, this control may be used any place where paged data is used.

Selector: hc-pagination

Properties
Name Description
@Input()

pageNumber: number | null

The current page number (one-based). Should be two-way bound using [(pageNumber)] to ensure that the source property is updated when the control is used.
@Input()

totalPages: number | null | undefined

The total pages possible to navigate to
@Output()

pageNumberChange: new EventEmitter<number | null>()

Event for when page number changes
